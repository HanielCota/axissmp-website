# AxisSMP Website

Website oficial do servidor de Minecraft AxisSMP.

## Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Estrutura

```
├── app/              # Rotas (App Router)
│   ├── news/         # Notícias
│   ├── rules/        # Regras
│   └── wiki/         # Wiki
├── components/       # Componentes React
│   ├── landing/      # Seções da landing
│   ├── layout/       # Layout (Navbar, etc)
│   └── ui/           # Componentes de UI
├── hooks/            # Custom hooks
├── lib/              # Utilitários e dados
├── types/            # TypeScript types
└── public/           # Assets estáticos
```

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
